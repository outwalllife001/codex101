快速搭建 React 项目

初级 10 分钟

目标

使用 Codex 从零创建一个 React + TypeScript + Vite 项目，包含路由、状态管理和基础 UI。

步骤 1：初始化项目

启动 Codex 并输入：

> 创建一个 React + TypeScript + Vite 项目，项目名为 my-app。
> 使用 React Router 配置路由，包含首页、关于页和 404 页面。
> 使用 Tailwind CSS 进行样式设置。
> 添加一个全局状态管理（使用 Zustand）。

步骤 2：等待执行

Codex 会：

  1. 运行 npm create vite@latest
  2. 安装依赖（react-router-dom, zustand, tailwindcss）
  3. 配置 Tailwind
  4. 创建路由文件
  5. 创建页面组件
  6. 设置 Zustand store

步骤 3：验证结果

Codex 完成后，运行：

cd my-app
npm run dev

生成的项目结构

my-app/
├── src/
│   ├── components/      # 共享组件
│   ├── pages/          # 页面组件
│   │   ├── Home.tsx
│   │   ├── About.tsx
│   │   └── NotFound.tsx
│   ├── store/          # Zustand store
│   │   └── index.ts
│   ├── App.tsx         # 路由配置
│   └── main.tsx
├── index.html
├── package.json
├── tailwind.config.js
└── vite.config.ts

关键学习点

  • 一次提示完成多步骤任务
  • 自动处理依赖安装
  • 生成标准项目结构
  • 遵循最佳实践（hooks、类型定义）

React プロジェクトを素早く構築

初級 10分

目的

Codex を使って、ルーティング、状態管理、基本 UI を備えた React + TypeScript + Vite プロジェクトをゼロから作成します。

ステップ 1: プロジェクトを初期化する

Codex を起動して、次のように入力します。

> React + TypeScript + Vite のプロジェクトを作成してください。プロジェクト名は my-app。
> React Router でルーティングを設定し、Home、About、404 ページを含めてください。
> Tailwind CSS でスタイリングを設定してください。
> Zustand を使ったグローバル状態管理を追加してください。

ステップ 2: 実行を待つ

Codex は次の処理を行います。

  1. npm create vite@latest を実行
  2. 依存関係をインストール（react-router-dom、zustand、tailwindcss）
  3. Tailwind を設定
  4. ルーティング関連ファイルを作成
  5. ページコンポーネントを作成
  6. Zustand ストアをセットアップ

ステップ 3: 結果を確認する

Codex の処理が完了したら、次を実行します。

cd my-app
npm run dev

生成されるプロジェクト構成

my-app/
├── src/
│   ├── components/      # 共通コンポーネント
│   ├── pages/          # ページコンポーネント
│   │   ├── Home.tsx
│   │   ├── About.tsx
│   │   └── NotFound.tsx
│   ├── store/          # Zustand ストア
│   │   └── index.ts
│   ├── App.tsx         # ルーティング設定
│   └── main.tsx
├── index.html
├── package.json
├── tailwind.config.js
└── vite.config.ts

学べるポイント

  • 1 回のプロンプトで複数ステップの作業を完了できる
  • 依存関係のインストールを自動化できる
  • 標準的なプロジェクト構造をそのまま生成できる
  • hooks や型定義などのベストプラクティスに従える

Crear un proyecto React rapidamente

Principiante 10 min

Objetivo

Usa Codex para crear desde cero un proyecto React + TypeScript + Vite con enrutamiento, gestion de estado y una UI basica.

Paso 1: Inicializar el proyecto

Inicia Codex e introduce lo siguiente:

> Crea un proyecto React + TypeScript + Vite llamado my-app.
> Configura el enrutamiento con React Router e incluye una pagina de inicio, una pagina About y una pagina 404.
> Configura los estilos con Tailwind CSS.
> Agrega gestion de estado global con Zustand.

Paso 2: Esperar la ejecucion

Codex hara lo siguiente:

  1. Ejecutar npm create vite@latest
  2. Instalar dependencias (react-router-dom, zustand, tailwindcss)
  3. Configurar Tailwind
  4. Crear archivos de rutas
  5. Crear componentes de pagina
  6. Preparar el store de Zustand

Paso 3: Verificar el resultado

Cuando Codex termine, ejecuta:

cd my-app
npm run dev

Estructura del proyecto generado

my-app/
├── src/
│   ├── components/      # Componentes compartidos
│   ├── pages/          # Componentes de pagina
│   │   ├── Home.tsx
│   │   ├── About.tsx
│   │   └── NotFound.tsx
│   ├── store/          # Store de Zustand
│   │   └── index.ts
│   ├── App.tsx         # Configuracion de rutas
│   └── main.tsx
├── index.html
├── package.json
├── tailwind.config.js
└── vite.config.ts

Puntos clave de aprendizaje

  • Resolver una tarea de varios pasos con un solo prompt
  • Automatizar la instalacion de dependencias
  • Generar una estructura de proyecto estandar
  • Seguir buenas practicas como hooks y definiciones de tipos
