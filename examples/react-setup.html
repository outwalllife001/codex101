快速搭建 React 项目

初级 10 分钟

目标

使用 Codex 从零创建一个 React + TypeScript + Vite 项目，包含路由、状态管理和基础 UI。

步骤 1：初始化项目

启动 Codex 并输入：

> 创建一个 React + TypeScript + Vite 项目，项目名为 my-app。
> 使用 React Router 配置路由，包含首页、关于页和 404 页面。
> 使用 Tailwind CSS 进行样式设置。
> 添加一个全局状态管理（使用 Zustand）。

步骤 2：等待执行

Codex 会：

  1. 运行 npm create vite@latest
  2. 安装依赖（react-router-dom, zustand, tailwindcss）
  3. 配置 Tailwind
  4. 创建路由文件
  5. 创建页面组件
  6. 设置 Zustand store

步骤 3：验证结果

Codex 完成后，运行：

cd my-app
npm run dev

生成的项目结构

my-app/
├── src/
│   ├── components/      # 共享组件
│   ├── pages/          # 页面组件
│   │   ├── Home.tsx
│   │   ├── About.tsx
│   │   └── NotFound.tsx
│   ├── store/          # Zustand store
│   │   └── index.ts
│   ├── App.tsx         # 路由配置
│   └── main.tsx
├── index.html
├── package.json
├── tailwind.config.js
└── vite.config.ts

关键学习点

  • ✅ 一次提示完成多步骤任务
  • ✅ 自动处理依赖安装
  • ✅ 生成标准项目结构
  • ✅ 遵循最佳实践（hooks、类型定义）
