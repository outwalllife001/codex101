快速搭建 React 项目

初级 10 分钟

目标

使用 Codex 从零创建一个 React + TypeScript + Vite 项目，包含路由、状态管理和基础 UI。

步骤 1：初始化项目

启动 Codex 并输入：

> 创建一个 React + TypeScript + Vite 项目，项目名为 my-app。
> 使用 React Router 配置路由，包含首页、关于页和 404 页面。
> 使用 Tailwind CSS 进行样式设置。
> 添加一个全局状态管理（使用 Zustand）。

步骤 2：等待执行

Codex 会：

  1. 运行 npm create vite@latest
  2. 安装依赖（react-router-dom, zustand, tailwindcss）
  3. 配置 Tailwind
  4. 创建路由文件
  5. 创建页面组件
  6. 设置 Zustand store

步骤 3：验证结果

Codex 完成后，运行：

cd my-app
npm run dev

生成的项目结构

my-app/
├── src/
│   ├── components/      # 共享组件
│   ├── pages/          # 页面组件
│   │   ├── Home.tsx
│   │   ├── About.tsx
│   │   └── NotFound.tsx
│   ├── store/          # Zustand store
│   │   └── index.ts
│   ├── App.tsx         # 路由配置
│   └── main.tsx
├── index.html
├── package.json
├── tailwind.config.js
└── vite.config.ts

关键学习点

  • 一次提示完成多步骤任务
  • 自动处理依赖安装
  • 生成标准项目结构
  • 遵循最佳实践（hooks、类型定义）

Build a React Project

Beginner 10 min

Goal

Use Codex to create a React + TypeScript + Vite project from scratch, including routing, state management, and basic UI.

Step 1: Initialize the Project

Launch Codex and enter:

> Create a React + TypeScript + Vite project named my-app.
> Configure routing with React Router, including a home page, about page, and 404 page.
> Set up styling with Tailwind CSS.
> Add global state management (using Zustand).

Step 2: Wait for Execution

Codex will:

  1. Run npm create vite@latest
  2. Install dependencies (react-router-dom, zustand, tailwindcss)
  3. Configure Tailwind
  4. Create routing files
  5. Create page components
  6. Set up Zustand store

Step 3: Verify the Result

After Codex finishes, run:

cd my-app
npm run dev

Generated Project Structure

my-app/
├── src/
│   ├── components/      # Shared components
│   ├── pages/          # Page components
│   │   ├── Home.tsx
│   │   ├── About.tsx
│   │   └── NotFound.tsx
│   ├── store/          # Zustand store
│   │   └── index.ts
│   ├── App.tsx         # Routing config
│   └── main.tsx
├── index.html
├── package.json
├── tailwind.config.js
└── vite.config.ts

Key Takeaways

  • Complete multi-step tasks with a single prompt
  • Automatic dependency installation
  • Generates standard project structure
  • Follows best practices (hooks, type definitions)
