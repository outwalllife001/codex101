# Codex 学院 - OpenAI Codex 完全学习指南

一个系统化的 OpenAI Codex 学习网站，包含从入门到精进的完整内容。

> 本项目是开源项目，欢迎贡献！GitHub 仓库：[outwalllife001/codex101](https://github.com/outwalllife001/codex101)

## 📚 内容结构

### 学习指南 (guides/)
- **installation.html** - 安装与配置
- **concepts.html** - 核心概念（Agent 模式、审批系统、上下文管理）
- **basic-usage.html** - 基础用法
- **slash-commands.html** - 24+ Slash 命令完整参考
- **agents-md.html** - AGENTS.md 完全指南
- **skills.html** - Skills 扩展开发
- **mcp.html** - MCP 集成指南
- **advanced.html** - 高级技巧（提示工程、多 Agent 协作）

### 实战案例 (examples/)
- **react-setup** - 快速搭建 React 项目（初级）
- **api-backend** - API 后端开发（中级）
- **refactoring** - 代码重构与优化（高级）
- **multi-agent** - 多 Agent 协作开发（高级）

### 资源下载 (downloads/)
- **agents-md-template.md** - AGENTS.md 模板
- **config-example.toml** - 配置文件示例
