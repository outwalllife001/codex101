# Codex アカデミー - OpenAI Codex 完全学習ガイド

入門から上級まで体系的に学べる OpenAI Codex 学習サイトです。

> 本プロジェクトはオープンソースです。コントリビューション歓迎！GitHub リポジトリ：[outwalllife001/codex101](https://github.com/outwalllife001/codex101)

## コンテンツ構成

### 学習ガイド (guides/)
- **installation.html** - インストールと設定
- **concepts.html** - コアコンセプト（Agent モード、承認システム、コンテキスト管理）
- **basic-usage.html** - 基本的な使い方
- **slash-commands.html** - 24以上の Slash コマンド完全リファレンス
- **agents-md.html** - AGENTS.md 完全ガイド
- **skills.html** - Skills 拡張開発
- **mcp.html** - MCP 統合ガイド
- **advanced.html** - 上級テクニック（プロンプトエンジニアリング、マルチ Agent 連携）

### 実践例 (examples/)
- **react-setup** - React プロジェクトの素早い構築（初級）
- **api-backend** - API バックエンド開発（中級）
- **refactoring** - コードリファクタリングと最適化（上級）
- **multi-agent** - マルチ Agent 共同開発（上級）

### リソースダウンロード (downloads/)
- **agents-md-template.md** - AGENTS.md テンプレート
- **config-example.toml** - 設定ファイルサンプル

## クイックスタート

ブラウザで `index.html` を開くと、すべてのコンテンツを閲覧できます。

1. **入門** → インストールと設定 → コアコンセプト
2. **使い方の習得** → 基本的な使い方 → Slash コマンド

4. **ツール統合** → MCP → 上級テクニック
5. **実践演習** → ケーススタディ
