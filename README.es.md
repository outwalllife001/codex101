# Codex Academy - Guia Completa de Aprendizaje de OpenAI Codex

Un sitio web de aprendizaje sistematico de OpenAI Codex, que cubre desde lo basico hasta tecnicas avanzadas.

> Este es un proyecto de codigo abierto. Las contribuciones son bienvenidas! Repositorio en GitHub: [outwalllife001/codex101](https://github.com/outwalllife001/codex101)

## Estructura del Contenido

### Guias de Aprendizaje (guides/)
- **installation.html** - Instalacion y Configuracion
- **concepts.html** - Conceptos Fundamentales (Modo Agent, Sistema de Aprobacion, Gestion de Contexto)
- **basic-usage.html** - Uso Basico
- **slash-commands.html** - Referencia Completa de 24+ Comandos Slash
- **agents-md.html** - Guia Completa de AGENTS.md
- **skills.html** - Desarrollo de Extensiones Skills
- **mcp.html** - Guia de Integracion MCP
- **advanced.html** - Tecnicas Avanzadas (Ingenieria de Prompts, Colaboracion Multi-Agent)

### Ejemplos Practicos (examples/)
- **react-setup** - Crear un Proyecto React Rapidamente (Principiante)
- **api-backend** - Desarrollo de Backend API (Intermedio)
- **refactoring** - Refactorizacion y Optimizacion de Codigo (Avanzado)
- **multi-agent** - Desarrollo Colaborativo Multi-Agent (Avanzado)

### Descargas de Recursos (downloads/)
- **agents-md-template.md** - Plantilla de AGENTS.md
- **config-example.toml** - Ejemplo de Archivo de Configuracion

## Inicio Rapido

Abre `index.html` en tu navegador para explorar todo el contenido.

## Ruta de Aprendizaje

1. **Primeros Pasos** - Instalacion y Configuracion - Conceptos Fundamentales
2. **Dominar el Uso** - Uso Basico - Comandos Slash
3. **Personalizacion del Proyecto** - AGENTS.md - Skills
4. **Integracion de Herramientas** - MCP - Tecnicas Avanzadas
5. **Practica** - Estudios de Caso

## Audiencia Objetivo

- Desarrolladores que quieren aprender programacion asistida por IA
- Ingenieros que buscan mejorar su productividad en el desarrollo
- Equipos tecnicos que necesitan una referencia rapida de Codex
