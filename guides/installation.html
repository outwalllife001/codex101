安装与配置

OpenAI Codex 提供多种使用方式：命令行界面 (CLI)、IDE 扩展（VS Code、Cursor、Windsurf）以及 Web 应用。 本指南将帮助你完成 Codex CLI 的安装和配置。

系统要求

  • 操作系统: macOS (Apple Silicon 或 Intel), Linux (x86_64 或 arm64)
  • Node.js: v18.0.0 或更高版本（用于 npm 安装）
  • Git: 用于代码版本控制
注意: Windows 支持目前处于实验阶段。建议在 WSL (Windows Subsystem for Linux) 中使用 Codex CLI。

安装方法

方法 1: 使用 npm 安装（推荐）

全局安装 Codex CLI：

npm install -g @openai/codex

方法 2: 使用 Homebrew 安装（macOS）

brew install --cask codex

方法 3: 下载二进制文件

从 GitHub Releases 下载适合你平台的二进制文件：

平台 架构 文件名
macOS Apple Silicon codex-aarch64-apple-darwin.tar.gz
macOS Intel codex-x86_64-apple-darwin.tar.gz
Linux x86_64 codex-x86_64-unknown-linux-musl.tar.gz
Linux ARM64 codex-aarch64-unknown-linux-musl.tar.gz

首次运行

安装完成后，在终端中运行：

codex

首次启动时，Codex 会提示你登录。你可以选择以下两种方式：

方式 1: 使用 ChatGPT 账号登录

推荐使用此方式，它将使用你现有的 ChatGPT Plus/Pro/Business 订阅。

  1. 运行 codex
  2. 选择 "Sign in with ChatGPT"
  3. 在浏览器中完成授权
  4. 返回终端，开始工作

方式 2: 使用 API Key

如果你更喜欢按量付费，可以使用 OpenAI API Key：

  1. 在 OpenAI 平台 创建 API Key
  2. 运行 codex
  3. 选择 "Sign in with API Key"
  4. 输入你的 API Key
重要提示: 使用 API Key 时某些功能（如云线程）可能不可用。

配置文件

Codex 的配置存储在 ~/.codex/config.toml 文件中。你可以手动编辑或使用命令行配置。

基本配置示例

# 核心模型设置
model = "gpt-5.4"
model_provider = "openai"

# 推理力度: minimal | low | medium | high | xhigh
model_reasoning_effort = "medium"

# 审批策略
approval_policy = "on-request"

# 沙箱模式: read-only | workspace-write | danger-full-access
sandbox_mode = "read-only"

# Web 搜索模式: disabled | cached | live
web_search = "cached"

# 文件打开器: vscode | vscode-insiders | windsurf | cursor | none
file_opener = "vscode"

验证安装

运行以下命令验证安装是否成功：

# 检查版本
codex --version

# 查看帮助
codex --help

# 检查会话状态（在交互模式下）
/status

升级 Codex

保持 Codex 更新以获得最新功能和修复：

# npm 安装方式
npm i -g @openai/codex@latest

# Homebrew 安装方式
brew upgrade codex

下一步

安装完成后，你可以：

  • 阅读 核心概念 了解 Codex 的工作原理
  • 查看 基础用法 学习基本操作
  • 浏览 实战案例 开始你的第一个项目

Installation & Configuration

OpenAI Codex offers multiple ways to use it: command-line interface (CLI), IDE extensions (VS Code, Cursor, Windsurf), and web applications. This guide will help you complete the installation and configuration of Codex CLI.

System Requirements

  • Operating System: macOS (Apple Silicon or Intel), Linux (x86_64 or arm64)
  • Node.js: v18.0.0 or higher (for npm installation)
  • Git: For code version control
Note: Windows support is currently experimental. It is recommended to use Codex CLI within WSL (Windows Subsystem for Linux).

Installation Methods

Method 1: Install via npm (Recommended)

Install Codex CLI globally:

npm install -g @openai/codex

Method 2: Install via Homebrew (macOS)

brew install --cask codex

Method 3: Download Binary

Download the binary for your platform from GitHub Releases:

Platform Architecture Filename
macOS Apple Silicon codex-aarch64-apple-darwin.tar.gz
macOS Intel codex-x86_64-apple-darwin.tar.gz
Linux x86_64 codex-x86_64-unknown-linux-musl.tar.gz
Linux ARM64 codex-aarch64-unknown-linux-musl.tar.gz

First Run

After installation, run in the terminal:

codex

On first launch, Codex will prompt you to log in. You can choose one of the following two methods:

Method 1: Sign in with ChatGPT Account

This is the recommended method, which uses your existing ChatGPT Plus/Pro/Business subscription.

  1. Run codex
  2. Select "Sign in with ChatGPT"
  3. Complete authorization in the browser
  4. Return to the terminal and start working

Method 2: Use API Key

If you prefer pay-as-you-go, you can use an OpenAI API Key:

  1. Create an API Key on the OpenAI Platform
  2. Run codex
  3. Select "Sign in with API Key"
  4. Enter your API Key
Important: Some features (such as cloud threads) may not be available when using an API Key.

# File opener: vscode | vscode-insiders | windsurf | cursor | none

Verify Installation

Run the following commands to verify the installation:

# View help

# Check session status (in interactive mode)

Upgrade Codex

Keep Codex updated to get the latest features and fixes:

インストールと設定

OpenAI Codex は、コマンドラインインターフェース（CLI）、IDE 拡張（VS Code、Cursor、Windsurf）、Web アプリなど複数の利用方法を提供しています。 このガイドでは、Codex CLI のインストールと初期設定を完了するまでを案内します。

システム要件

  • 対応 OS: macOS（Apple Silicon または Intel）、Linux（x86_64 または arm64）
  • Node.js: v18.0.0 以上（npm インストール時に必要）
  • Git: コードのバージョン管理に必要
注意: Windows 対応は現在実験段階です。Codex CLI は WSL（Windows Subsystem for Linux）内で使うことをおすすめします。

インストール方法

方法 1: npm でインストール（推奨）

Codex CLI をグローバルにインストールします：

npm install -g @openai/codex

方法 2: Homebrew でインストール（macOS）

brew install --cask codex

方法 3: バイナリをダウンロード

GitHub Releases から、利用中のプラットフォームに合ったバイナリを取得します：

プラットフォーム アーキテクチャ ファイル名
macOS Apple Silicon codex-aarch64-apple-darwin.tar.gz
macOS Intel codex-x86_64-apple-darwin.tar.gz
Linux x86_64 codex-x86_64-unknown-linux-musl.tar.gz
Linux ARM64 codex-aarch64-unknown-linux-musl.tar.gz

初回起動

インストール後、ターミナルで次を実行します：

codex

初回起動時にはログインが求められます。以下の 2 つの方法から選べます：

方法 1: ChatGPT アカウントでサインイン

既存の ChatGPT Plus/Pro/Business サブスクリプションを使えるため、この方法が推奨です。

  1. codex を実行する
  2. 「Sign in with ChatGPT」を選ぶ
  3. ブラウザで認証を完了する
  4. ターミナルに戻って作業を始める

方法 2: API Key を使う

従量課金で使いたい場合は OpenAI API Key を利用できます：

  1. OpenAI Platform で API Key を作成する
  2. codex を実行する
  3. 「Sign in with API Key」を選ぶ
  4. API Key を入力する
重要: API Key を使う場合、一部機能（クラウドスレッドなど）が利用できないことがあります。

Instalacion y Configuracion

OpenAI Codex ofrece varias formas de uso: interfaz de linea de comandos (CLI), extensiones de IDE (VS Code, Cursor, Windsurf) y aplicaciones web. Esta guia te ayudara a completar la instalacion y configuracion inicial de Codex CLI.

Requisitos del sistema

  • Sistema operativo: macOS (Apple Silicon o Intel), Linux (x86_64 o arm64)
  • Node.js: v18.0.0 o superior (necesario para instalar con npm)
  • Git: necesario para control de versiones
Nota: El soporte para Windows sigue siendo experimental. Se recomienda usar Codex CLI dentro de WSL (Windows Subsystem for Linux).

Metodos de instalacion

Metodo 1: Instalar con npm (Recomendado)

Instala Codex CLI de forma global:

npm install -g @openai/codex

Metodo 2: Instalar con Homebrew (macOS)

brew install --cask codex

Metodo 3: Descargar el binario

Descarga el binario adecuado para tu plataforma desde GitHub Releases:

Plataforma Arquitectura Nombre de archivo
macOS Apple Silicon codex-aarch64-apple-darwin.tar.gz
macOS Intel codex-x86_64-apple-darwin.tar.gz
Linux x86_64 codex-x86_64-unknown-linux-musl.tar.gz
Linux ARM64 codex-aarch64-unknown-linux-musl.tar.gz

Primer inicio

Despues de instalarlo, ejecuta esto en tu terminal:

codex

La primera vez que lo abras, Codex te pedira iniciar sesion. Puedes elegir una de estas dos opciones:

Metodo 1: Iniciar sesion con tu cuenta de ChatGPT

Este es el metodo recomendado porque aprovecha tu suscripcion existente de ChatGPT Plus/Pro/Business.

  1. Ejecuta codex
  2. Selecciona "Sign in with ChatGPT"
  3. Completa la autorizacion en el navegador
  4. Vuelve a la terminal y empieza a trabajar

Metodo 2: Usar API Key

Si prefieres pagar por uso, puedes iniciar con una OpenAI API Key:

  1. Crea una API Key en OpenAI Platform
  2. Ejecuta codex
  3. Selecciona "Sign in with API Key"
  4. Introduce tu API Key
Importante: Algunas funciones (como los hilos en la nube) pueden no estar disponibles cuando usas API Key.
