安装与配置

OpenAI Codex 提供多种使用方式：命令行界面 (CLI)、IDE 扩展（VS Code、Cursor、Windsurf）以及 Web 应用。 本指南将帮助你完成 Codex CLI 的安装和配置。

系统要求

  • 操作系统: macOS (Apple Silicon 或 Intel), Linux (x86_64 或 arm64)
  • Node.js: v18.0.0 或更高版本（用于 npm 安装）
  • Git: 用于代码版本控制
注意: Windows 支持目前处于实验阶段。建议在 WSL (Windows Subsystem for Linux) 中使用 Codex CLI。

安装方法

方法 1: 使用 npm 安装（推荐）

全局安装 Codex CLI：

npm install -g @openai/codex

方法 2: 使用 Homebrew 安装（macOS）

brew install --cask codex

方法 3: 下载二进制文件

从 GitHub Releases 下载适合你平台的二进制文件：

平台 架构 文件名
macOS Apple Silicon codex-aarch64-apple-darwin.tar.gz
macOS Intel codex-x86_64-apple-darwin.tar.gz
Linux x86_64 codex-x86_64-unknown-linux-musl.tar.gz
Linux ARM64 codex-aarch64-unknown-linux-musl.tar.gz

首次运行

安装完成后，在终端中运行：

codex

首次启动时，Codex 会提示你登录。你可以选择以下两种方式：

方式 1: 使用 ChatGPT 账号登录

推荐使用此方式，它将使用你现有的 ChatGPT Plus/Pro/Business 订阅。

  1. 运行 codex
  2. 选择 "Sign in with ChatGPT"
  3. 在浏览器中完成授权
  4. 返回终端，开始工作

方式 2: 使用 API Key

如果你更喜欢按量付费，可以使用 OpenAI API Key：

  1. 在 OpenAI 平台 创建 API Key
  2. 运行 codex
  3. 选择 "Sign in with API Key"
  4. 输入你的 API Key
重要提示: 使用 API Key 时某些功能（如云线程）可能不可用。

配置文件

Codex 的配置存储在 ~/.codex/config.toml 文件中。你可以手动编辑或使用命令行配置。

基本配置示例

# 核心模型设置
model = "gpt-5.4"
model_provider = "openai"

# 推理力度: minimal | low | medium | high | xhigh
model_reasoning_effort = "medium"

# 审批策略
approval_policy = "on-request"

# 沙箱模式: read-only | workspace-write | danger-full-access
sandbox_mode = "read-only"

# Web 搜索模式: disabled | cached | live
web_search = "cached"

# 文件打开器: vscode | vscode-insiders | windsurf | cursor | none
file_opener = "vscode"

验证安装

运行以下命令验证安装是否成功：

# 检查版本
codex --version

# 查看帮助
codex --help

# 检查会话状态（在交互模式下）
/status

升级 Codex

保持 Codex 更新以获得最新功能和修复：

# npm 安装方式
npm i -g @openai/codex@latest

# Homebrew 安装方式
brew upgrade codex
