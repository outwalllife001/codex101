基础用法

本节介绍 Codex CLI 的基本操作，帮助你快速上手使用 Codex 进行 AI 辅助编程。

启动 Codex

在项目目录中运行：

codex

这将启动交互式 TUI（文本用户界面）。

第一个任务

进入 Codex 后，直接用自然语言描述你想做的事情：

> 创建一个简单的 HTTP 服务器，监听 3000 端口，返回 "Hello World"

理解响应

Codex 会：

  1. 分析你的请求
  2. 提出执行计划（列出要创建/修改的文件）
  3. 等待你确认（按 Enter）
  4. 执行并显示结果

常用操作模式

1. 代码生成

> 写一个函数，接收日期字符串，返回该日期是星期几

2. 代码解释

> 解释这个文件的作用 @src/utils/helper.ts

3. 代码重构

> 将这个回调函数重构为 async/await 形式

4. 错误修复

> 这个测试失败了，请修复：@tests/auth.test.ts

文件引用

使用 @ 符号引用文件或目录：

  • @filename.ts - 引用单个文件
  • @src/ - 引用整个目录
  • @README.md @package.json - 引用多个文件

上下文管理

查看状态

/status

压缩上下文（长会话）

/compact

开始新会话

/new

退出 Codex

/quit

或按 Ctrl+C

Basic Usage

This section introduces the basic operations of Codex CLI, helping you quickly get started with AI-assisted programming using Codex.

Launch Codex

Run in your project directory:

codex

This will launch the interactive TUI (Text User Interface).

Your First Task

Once inside Codex, describe what you want to do in natural language:

> Create a simple HTTP server that listens on port 3000 and returns "Hello World"

Understanding the Response

Codex will:

  1. Analyze your request
  2. Propose an execution plan (listing files to create/modify)
  3. Wait for your confirmation (press Enter)
  4. Execute and display the results

Common Operation Modes

1. Code Generation

> Write a function that takes a date string and returns the day of the week

2. Code Explanation

> Explain what this file does @src/utils/helper.ts

3. Code Refactoring

> Refactor this callback function to use async/await

4. Bug Fixing

> This test is failing, please fix it: @tests/auth.test.ts

File References

Use the @ symbol to reference files or directories:

  • @filename.ts - Reference a single file
  • @src/ - Reference an entire directory
  • @README.md @package.json - Reference multiple files

Context Management

Check Status

/status

Compact Context (Long Sessions)

/compact

Start a New Session

/new

Exit Codex

/quit

Or press Ctrl+C

基本的な使い方

このセクションでは Codex CLI の基本操作を紹介します。Codex を使った AI 支援プログラミングを素早く始めるための土台になります。

Codex を起動する

プロジェクトのディレクトリで次を実行します：

codex

これで対話型 TUI（Text User Interface）が起動します。

最初のタスク

Codex に入ったら、やりたいことを自然言語で説明します：

> 3000 番ポートで待ち受けて "Hello World" を返すシンプルな HTTP サーバーを作成して

応答の流れを理解する

Codex は次のように動作します：

  1. 要求内容を解析する
  2. 実行プランを提案する（作成・変更するファイルを一覧化）
  3. 確認を待つ（Enter キーで承認）
  4. 実行して結果を表示する

代表的な操作モード

1. コード生成

> 日付文字列を受け取って曜日を返す関数を書いて

2. コード説明

> このファイルが何をしているか説明して @src/utils/helper.ts

3. リファクタリング

> このコールバック関数を async/await に書き換えて

4. バグ修正

> このテストが失敗しています。修正してください: @tests/auth.test.ts

ファイル参照

@ 記号を使ってファイルやディレクトリを参照できます：

  • @filename.ts - 単一ファイルを参照
  • @src/ - ディレクトリ全体を参照
  • @README.md @package.json - 複数ファイルを同時に参照

Uso Basico

Esta seccion presenta las operaciones basicas de Codex CLI y te ayuda a empezar rapidamente con programacion asistida por IA usando Codex.

Iniciar Codex

Ejecuta esto dentro del directorio de tu proyecto:

codex

Esto abrira la TUI interactiva (Text User Interface).

Tu primera tarea

Una vez dentro de Codex, describe en lenguaje natural lo que quieres hacer:

> Crea un servidor HTTP simple que escuche en el puerto 3000 y devuelva "Hello World"

Entender la respuesta

Codex hara lo siguiente:

  1. Analizar tu solicitud
  2. Proponer un plan de ejecucion (listando los archivos a crear o modificar)
  3. Esperar tu confirmacion (pulsa Enter)
  4. Ejecutar y mostrar los resultados

Modos de operacion comunes

1. Generacion de codigo

> Escribe una funcion que reciba una cadena de fecha y devuelva el dia de la semana

2. Explicacion de codigo

> Explica que hace este archivo @src/utils/helper.ts

3. Refactorizacion de codigo

> Refactoriza esta funcion con callback para usar async/await

4. Correccion de errores

> Esta prueba esta fallando, por favor corrigela: @tests/auth.test.ts

Referencias de archivos

Usa el simbolo @ para referenciar archivos o directorios:

  • @filename.ts - Referenciar un solo archivo
  • @src/ - Referenciar un directorio completo
  • @README.md @package.json - Referenciar varios archivos

Compactar contexto (sesiones largas)

/compact

Iniciar una nueva sesion

Salir de Codex

/quit
