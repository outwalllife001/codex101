Slash 命令完整参考

Slash 命令是 Codex CLI 中快速控制会话的方式。在交互式 TUI 中输入 / 即可打开命令列表。 本页面列出了所有 24+ 个内置命令及其使用场景。

高频使用流程

/status          # 检查当前配置
/model          # 切换模型
/permissions    # 设置权限
/mention <文件>  # 引用关键文件
/plan           # 制定执行计划
/review         # 代码审查
/diff           # 查看变更
/compact        # 压缩上下文

会话控制命令

命令 用途 使用场景
/new 开始新会话 清理上下文，保持在同一 CLI 进程中
/resume 恢复历史会话 打开选择器显示最近的会话列表
/fork 克隆当前会话 复制对话到新线程，尝试替代方案不丢失进度
/quit / /exit 退出 CLI 完全退出 Codex CLI
/compact 压缩上下文 总结对话以释放 token 空间，避免达到上下文限制

模型和样式命令

命令 用途 使用场景
/model 切换模型和推理级别 交互式选择，可在会话中途切换
/personality 更改沟通风格 可选: friendly / pragmatic / none
/plan 进入规划模式 Codex 在执行前起草计划，适合复杂任务

权限和状态命令

命令 用途 使用场景
/permissions 调整授权模式 在 Auto / Read Only / Full Access 之间切换
/status 查看会话信息 显示模型、token 使用量和账户详情
/statusline 自定义状态栏 交互式调整底部状态栏显示内容
/debug-config 调试配置 打印完整的配置加载链和策略诊断

代码和审查命令

命令 用途 使用场景
/review 代码审查 审查工作目录的变更，或对比基础分支进行 PR 式审查
/diff 查看 Git diff 显示 Git diff，包括未跟踪的文件
/undo 撤销最近操作 回滚 Codex 最近的一次编辑或命令执行
/mention 引用文件 将特定文件或文件夹附加到对话中作为上下文

工具和集成命令

命令 用途 使用场景
/mcp 列出 MCP 工具 查看配置的 Model Context Protocol 服务器和工具
/apps 浏览应用 浏览 ChatGPT 应用(连接器)并插入到提示中
/agent 切换 Agent 线程 在并行运行的多个 Agent 之间切换
/skills 列出技能 查看可用技能或调用特定技能

实用命令

命令 用途 使用场景
/init 初始化 AGENTS.md 在当前目录生成 AGENTS.md 脚手架文件
/clear 清屏 清空终端并开始新的聊天
/copy 复制输出 复制 Codex 最新的完整输出或计划文本

使用技巧

1. 使用 /fork 进行方案对比

/fork 命令非常实用。当你实现功能时想尝试另一种方案， 使用 fork 可以保留当前进度，同时在新线程中尝试替代方案。

2. 复杂任务前使用 /plan

对于"重构整个认证模块"这样的复杂任务，先用 /plan 让 Codex outlining 步骤。 审查并批准计划后再执行，这样更安全，避免浪费精力在错误的方法上。

3. 定期使用 /compact

不要等到上下文限制错误才使用 /compact。在长会话中定期压缩， 可以保持对话的清晰度，同时保留关键信息。

4. 使用 /debug-config 排查问题

当配置设置似乎没有生效时，/debug-config 可以显示 Codex 加载了哪些文件， 以及哪些值覆盖了其他值，非常有助于故障排查。

自定义命令

你可以在 ~/.codex/config.toml 中定义自定义 Slash 命令：

[[custom_commands]]
name = "test"
prompt = "运行测试套件并总结失败的测试"

[[custom_commands]]
name = "docs"
prompt = "更新当前编辑函数的文档"

[[custom_commands]]
name = "security"
prompt = "审查此代码的安全漏洞，重点关注输入验证和认证"

添加后，你可以直接在 Codex 中使用：

/test
/docs
/security
